== Description ==

TimelineJS is an open-source tool that enables you to build visually-rich interactive timelines and is available in 40 languages. This plugin lets you display you create timeline events and embed timelines within your posts. You can display your post categories as a timelime and add extra data to each of your posts, turning them into media-rich timeline events.

Features

*	Create fantastic looking Timelines within Wordpress, quickly and easily.
*	Embed Timelines within Posts and Pages effortlessly!
*	Timeline events and Timelines are easily created and managed.
*	Post Categories can be viewed as their own Timeline.
*	Posts can have extra data added to them to create a Timeline event.
*	All the features of TimelineJS3.

== Installation ==

StoryMaps is extremely easy to install and use.

1. Upload `timelines` to the `/wp-content/plugins/` directory & activate
1. Upload `advanced custom fields` or `advaned custom fields pro` to the `/wp-content/plugins/` directory & activate
1. Activate Timelines
1. Create a few 'Timeline Events'
1. Create a Timeline and add Timeline Events to it, order them if required.
1. View Timelines at its page URL or embed it into your post/page with this shortcode: `[timeline id="example-unqiue" name="basic-storymap-example"]`  (id is a unique div id, name is the timelines 'slug')
